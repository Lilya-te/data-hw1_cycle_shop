# Домашнее задание 1. Нормализация базы данных

При выполнении задания использовано:
1. База развернута в докер-контейнере.
2. Файл загружен и добавлен в контейнер.
3. С помощью командной строки терминала создана временная таблица и файл загружен в таблицу.
4. С помощью инструментов https://dbdiagram.io/ и функционала DBeaver разработана схема БД.
5. С помщью SQL запросов наполнены нормализованные таблицы.
6. Результат из таблиц собран в один запрос, который совпадает с изначальной таблицей.
